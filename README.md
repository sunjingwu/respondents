# 基于SlateJS可定位的答卷制作工具

## 概述
用户可以在线制作答卷，生成PDF文件同时提供位置信息，供CMR工具识别，做到自动批改等功能。

## 功能
- [ ] 定位

- [x] 分页

## 预览

```bash
$ npm install
$ npm start
```

or:

```bash
$ yarn
$ yarn start
```
